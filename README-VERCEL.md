# GlowSamer Aura - Vercel

Arquivos prontos para subir no Vercel.

## Como publicar

1. Envie esta pasta para o Vercel.
2. Escolha a opcao de projeto estatico.
3. Deixe os comandos de build vazios.
4. Use a pasta raiz do projeto como diretorio de publicacao.

O arquivo principal do site e `index.html`.
